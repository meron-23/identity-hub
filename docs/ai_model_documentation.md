# AI Model Documentation

Overview of biometric models used in Identity Hub.

## Face Detection
- Model: SSD / MTCNN
- Function: Localizes faces in video/images.

## Face Recognition
- Model: FaceNet / ArcFace
- Function: Generates 128/512 dimension embeddings.

## Liveness Detection
- Model: Custom RNN / Blink detection.
- Function: Prevents spoofing attacks.
