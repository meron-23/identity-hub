# Demo Video Script

**Objective**: Showcase the secure enrollment and biometric payment flow of Identity Hub.

## Scene 1: Enrollment
1. Show public landing page.
2. User enters details and takes a face photo.
3. System confirms DID generation success.

## Scene 2: Card Linking
1. User logs in with face scan.
2. User links a bank card from Mock Bank A.
3. Token is generated and stored securely.

## Scene 3: Payment
1. User goes to checkout.
2. User scans face to authorize payment.
3. Payment success message displayed.
