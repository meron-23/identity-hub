# Identity Hub

Identity Hub is a multi-modular project designed for Secure Identity Management, Biometrics, and Card Transactions.

## Project Structure

- `backend/`: Node.js Express API.
- `frontend/`: React-based User Interface.
- `ai/`: Python-based Biometric models.
- `card/`: Card simulation and tokenization.
- `docs/`: System documentation and diagrams.
- `demo/`: Demo scripts and assets.
- `scripts/`: Initialization and seeding scripts.

## How to Run

1. Run the setup script: `npm run setup`
2. Start the backend: `cd backend && npm start`
3. Start the frontend: `cd frontend && npm start`
